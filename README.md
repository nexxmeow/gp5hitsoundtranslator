# .gp5 to .osu Hitsound Translator

I made this just for practice and it obviously doesn't support multi-timing properly (i didn't even test that :D), so I recommend using it for songs that only have a single timing point.

Feel free to test and pull request your own fixes

## How to use:
 
So we need a guitar tab file with a drum track and a timed .osu file. Although it works only with .gp5 files, you still can convert .gpx or .gp files to .gp5 and it works just fine.

### Installation:

```
git clone https://github.com/nexxmeow/gp5hitsoundtranslator
pip install -r requirements.txt
```
